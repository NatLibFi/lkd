Linkitetyn kirjastodatan tietomalli (LKD) - BIBFRAME 2.2 -tietomallin kansallinen sovellus

Tämä BIBFRAME-tietomalli perustuu Kongressin kirjaston julkaiseman BIBFRAME-tietomallin 2.2.0 versioon. Mallia on täydennetty RDA Registryn luokilla ja ominaisuuksilla sekä suomalaisiin kansallisiin kuvailuohjeisiin perustuvilla täydennyksillä. Tämä 0.3.0-testiversio sisältää uudistuneen RDA:n mukaiset kansalliset suppean ja laajan kuvailutason elementit. Käytetyt tunnisteet ovat väliaikaisia, eikä malli ole valmis käyttöön. 0.2-sarjan versioihin nähden 0.3.0 sisältää uudet linkitysominaisuudet RDA:n suuntaan ja linkkien korjauksia, selkeytetyn nimekekokonaisuuden sekä joukon pienempiä korjauksia. Jätimme tästä versiosta toistaiseksi pois ruotsinkieliset nimekkeet, jotka on tarkoitus käydä läpi ja korjata systemaattisesti tulevaan versioon. Tarkoituksemme on julkaista vielä syksyn aikana tietomallin versio 0.3.1, joka sisältää teos- ja ekspressiojaon loppuunviennin. Uutena ominaisuutena Schema.finto.fi kykenee näyttämään yläluokalle määritellyt ominaisuudet alaluokan yhteydessä, joka helpottaa tietomallin tarkastelua.

Yksi merkittävä ero Kongressin kirjaston BIBFRAME-versioon nähden on, että LKD-tietomalli jakaa BIBFRAMEn Work-tason RDA:n mukaisiin teos- ja ekspressiotasoihin.

About the data model

This BIBFRAME data model is based on the BIBFRAME 2.2.0 data model published by the Library of Congress. The model has been supplemented with categories and properties from RDA Registry, as well as additions based on the Finnish description guidelines. This 0.3.0 test version comprises of the RDA elements determined by the Finnish cataloguing levels. The identifiers used are temporary and the model is not ready for use. Compared to the 0.2 versions of the data model, this one contains a new set of linking properties used for linking to RDA as well as corrections to said links, a more coherent approach to titles, and a host of minor corrections. We temporarily left Swedish labels out of this version as we intend to check and correct them systematically and introduce them back into a future version. We intend to release a further version 0.3.1 later this fall, which will conclude the division of bf:Work into lkd:Work and lkd:Expression. As a new feature, Schema.finto.fi is able to display properties associated with the superclass of a given entity.

A notable difference to the Library of Congress BIBFRAME data model is that LKD data model splits bf:Work into lkd:Work and lkd:Expression corresponding to the structure of RDA classes.
